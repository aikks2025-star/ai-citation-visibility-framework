# ai-citation-visibility-framework
How AI systems decide which businesses to cite. Reproducible measurement framework for AI visibility across ChatGPT, Claude, Perplexity, Google AI Overviews, and Microsoft Copilot. Includes the MAHI Index, MAD-M decay model, and MAHI-100 benchmark. By Kristina Shrider, ORCID 0009-0002-2655-4629 (AIMRI; Market Disruptors Agency).
