# AI Agent Brand Selection

**Definition:** AI agent brand selection is the process by which an AI system retrieves public signals, evaluates trust, freshness, topical fit, entity clarity, and corroboration, then chooses which brand to cite, recommend, or route a user toward.

No outside agency controls an AI agent's final choice. The practical work is improving the evidence layer that agents use to shortlist, cite, and recommend a brand.

## Why it matters

As AI agents handle more search, shopping, and comparison tasks, brands compete to become the trusted source or recommended option inside the answer, not only the highest organic link.

## Measurement method

Use MAHI Index dimensions to diagnose readiness: Entity Resolution, Citation Surface, Semantic Coverage, Freshness, and Trust Graph. Use MAHI-100-style prompt panels to verify whether the brand is cited, mentioned, omitted, or displaced by competitors.
